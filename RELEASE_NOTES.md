## Features

- **3D Globe Visualization**: Interactive globe.gl map showing ban origin countries with arc connections
- `/api/globe` endpoint: returns country-level ban statistics (total + active bans)
- GeoIP module: IP-to-country lookup via ip-api.com with thread-safe caching
- Responsive drawer panel (desktop 50% slide-in, mobile bottom sheet)
- Country labels show code + active/total ban count
- Active bans in red, inactive in blue
- Auto-refresh every 60 seconds
